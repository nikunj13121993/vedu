## Deploy to GitHub Pages

### Step 1: Create a GitHub Repository

1. Go to [GitHub](https://github.com) and log in
2. Click the "+" icon in the top right corner
3. Select "New repository"
4. Name it `vedu` (or any name you prefer)
5. Keep it **Public** (required for free GitHub Pages)
6. Click "Create repository"

### Step 2: Push Your Code

Open terminal in this folder and run:

```bash
git remote add origin https://github.com/YOUR_USERNAME/vedu.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click "Settings" tab
3. Scroll down to "Pages" in the left sidebar
4. Under "Source", select "main" branch
5. Click "Save"

### Step 4: Get Your Link

Your website will be live at:
```
https://YOUR_USERNAME.github.io/vedu/
```

It may take a few minutes to deploy the first time.

## Local Preview
